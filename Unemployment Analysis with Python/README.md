📉 Unemployment Analysis using Python


📌 Project Description

This project focuses on analyzing unemployment trends in India using data analysis techniques. It also highlights the impact of COVID-19 on employment rates across different states.

🎯 Objectives

- Analyze unemployment trends over time
- Study the impact of COVID-19
- Compare state-wise unemployment rates
- Identify patterns and relationships in data

📊 Dataset Features

- Region / State
- Date
- Estimated Unemployment Rate (%)
- Estimated Employed
- Labour Participation Rate (%)

🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

⚙️ Project Workflow

🔹 Data Preprocessing

- Handling missing values
- Converting Date column
- Removing duplicates

🔹 Exploratory Data Analysis (EDA)

- Distribution of unemployment rate
- Trend analysis

🔹 Time Series Analysis

- Visualizing unemployment over time

🔹 COVID-19 Impact Analysis

- Focus on year 2020
- Observed sharp increase in unemployment

🔹 State-wise Analysis

- Compared different states

🔹 Visualization

- Line plots
- Bar charts
- Heatmaps 

📈 Key Insights

- Unemployment rate increased significantly during COVID-19
- Lockdown had a major impact on employment
- Different states show different unemployment patterns
