
🚗 Car Price Prediction using Machine Learning

📌 Project Description

This project focuses on predicting car prices using Machine Learning regression techniques. The model learns patterns from historical data to estimate the selling price of cars.


🎯 Objectives

- Build a regression model
- Analyze factors affecting car prices
- Predict selling price accurately


📊 Dataset Features

- Car Name
- Year
- Selling Price (Target Variable)
- Present Price
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission
- Owner


🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

⚙️ Project Workflow

1. Data Cleaning
2. Feature Engineering
3. Model Training
4. Model Evaluation


📈 Results

- The model predicts car prices based on input features
- Regression techniques provide accurate predictions
